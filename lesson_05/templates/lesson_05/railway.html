How it works

  1. help — the agent calls action="help" to receive the live API documentation.
  2. Follow docs — action names and parameters are taken verbatim from the help response; nothing is guessed.
  3. 503 handling — automatic retry with exponential back-off (2 → 4 → 8 … s, capped at 120 s); up to 8 retries per call.
  4. Rate limits — after every successful response, X-RateLimit-* and Retry-After headers are inspected and honoured with an automatic sleep.
  5. Supervisor loop — a LangGraph supervisor checks for {FLG:…} in every response and terminates the graph once the flag is found.
  6. Artifacts — on completion, log.json and answer.json are saved to 0105task_context/.
